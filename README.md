# DevProfile — Generador de CV en PDF

## Descripción
Aplicación web en React para crear, editar y exportar CVs profesionales en PDF.
(proyecto universitario)

## Tecnologías
- React 19 + Vite
- React Router DOM
- Context API + localStorage
- Recharts (gráficas)
- jsPDF + html2canvas (exportación PDF)

## Instalación y ejecución
npm install
npm run dev
